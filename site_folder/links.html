resources

a miscellaneous dump of browser bookmarks.

art resources

  • film-grab.com
    • repository of film screengrabs, good for composition studies
  • mapcrunch.com
    • random google street view generator, good for environment studies
  • gamesfashionarchive.net
    • 3D viewer of miscellaneous character outfits from popular games
  • hikaku-sitatter.com
    • height comparison visualizer
  • ArtShield
    • watermarker to protect images against AI scrapers
  • Asaro Head
    • 3D viewer for planes of the face
  • Laurel Austin's Facial Expression References
    • random japanese blogpost about facial expression nuances
  • tosexamples.carrd.co
    • example terms of service for artists with commissions
  • Sinix's Youtube Channel
    • everything i've learned about digital painting has been from this man's videos

game dev resources

  • models-resource.com
    • website where you can download 3d character model rips from various games (special mention to the Metal Gear Solid: Peace Walker page, good resource for realistic low-poly characters)
  • ChipTone
    • free tool for creating game sound effects (CC0)
  • TK's Free Sound FX
    • free sound effects, found from the Fear and Hunger dev
  • freesound.org
    • repository of free user-created sounds (CC0 sounds included)
  • BitFontMaker2
    • free bitmap fonts
  • ZX Origins
    • free 8x8 bitmap fonts
  • polyhaven.com
    • 100% free textures, HDRIs, models (CC0)
  • ambientcg.com
    • 100% free assets (CC0)
  • 3dassets.one
    • tool to search for free assets over multiple asset-hosting sites (CC0 assets included)
  • shadertoy.com
    • repository of user-made visual shaders
  • Game Design Wiki by Rosa Carbó-Mascarell
    • resource for game dev careers. this was also made by the person who tweeted that Disco Elysium was another "white guy detective game" and it would've been better as a cozy witch simulator in the alps, so, uh, something to keep in mind

web dev resources

  • CSS Generators
    • miscellaneous tools for making css shapes/gradients
  • place-hold.it
    • generate a placeholder image with any dimensions
  • randoma11y.com
    • generate web-accessible color combinations

writing resources

  • behindthename.com
    • repository of etymologies and histories of names
  • fantasynamegenerators.com
    • godsent character name generator
  • magicbabynames.com
    • search a name, generate similar names
  • etymonline.com
    • repository of etymologies
  • reversedictionary.org
    • search a definition and find related words
  • ipa-reader.xyz
    • TTS pronunciation of words written in IPA

  • Thunderbird
    • free email manager
  • LibreOffice

  • YT-DLP

  • FFmpeg
    • command-line program that converts any file type to any other file type

  • cobalt.tools

  • sharklasers.com
    • free disposable email
  • justbeamit.com
    • a file transfer tool that I keep forgetting to use so maybe putting it on this list will help me remember it
